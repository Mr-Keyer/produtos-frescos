🌿 Horta do Povo
Horta do Povo é uma landing page desenvolvida para apresentar e divulgar a venda de legumes e vegetais frescos provenientes de fazendas locais.
O objetivo do projeto é simular um site simples de comércio de produtos orgânicos, com foco em uma navegação clara e apresentação dos produtos.
Este projeto foi criado como parte do meu aprendizado em desenvolvimento web e também para compor meu portfólio.
📌 Objetivo do Projeto
Criar uma landing page funcional que:
Apresente uma marca fictícia de produtos orgânicos
Mostre os benefícios dos produtos
Direcione o usuário para uma página de produtos
Exiba preços e itens disponíveis
Simule um pequeno site de vendas
🥬 Funcionalidades
O site possui duas páginas principais:
Página Inicial (Landing Page)
Apresenta:
Nome da marca Horta do Povo
Banner principal
Descrição do serviço
Benefícios dos produtos
Botão para acessar os produtos
Essa página funciona como a entrada principal do site.
Página de Produtos
Mostra alguns produtos disponíveis como:
Alface
Tomate
Cenoura
Pimentão
Abobrinhas
Brócolis
Cada produto possui:
Imagem
Nome
Opções de compra
Faixa de preços
🎯 Proposta do Projeto
A proposta é simular um site simples de uma pequena empresa ou produtor local que vende alimentos frescos online.
Este projeto pode evoluir futuramente com:
Sistema de carrinho
Backend
Cadastro de pedidos
Integração com banco de dados
Versão responsiva mais avançada
