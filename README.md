# 🌿 Horta do Povo

O **Horta do Povo** é um projeto de landing page criado para simular um site de venda de legumes e vegetais frescos.  
O objetivo principal é apresentar produtos orgânicos de forma simples, clara e acessível, permitindo que o usuário visualize os produtos disponíveis e suas faixas de preço.

Este projeto foi desenvolvido como parte do meu aprendizado em **desenvolvimento web** e também para compor meu portfólio.

---

## 📌 Objetivo do Projeto

O projeto tem como objetivo:

- Criar uma landing page funcional
- Apresentar uma marca fictícia de produtos naturais
- Mostrar benefícios dos produtos orgânicos
- Direcionar o usuário para uma página de produtos
- Exibir preços e itens disponíveis
- Simular um pequeno site de vendas online

---

## 🥬 Funcionalidades do Site

O site é composto por duas páginas principais.

### Página Inicial (Landing Page)

A página inicial apresenta:

- Nome da marca **Horta do Povo**
- Banner principal
- Descrição sobre os produtos
- Benefícios da compra de alimentos orgânicos
- Botão para acessar a página de produtos

Essa página funciona como a porta de entrada do site.

---

### Página de Produtos

A página de produtos apresenta alguns legumes e vegetais disponíveis para compra.

**Produtos exibidos:**

- Alface
- Tomate
- Cenoura
- Pimentão
- Abobrinhas
- Brócolis

Cada produto possui:

- Imagem ilustrativa
- Nome do produto
- Faixa de preços
- Diferentes opções de compra

---

## 🎯 Proposta do Projeto

A proposta do projeto é simular um site de uma pequena empresa ou produtor local que vende alimentos frescos pela internet.

Este projeto também demonstra conceitos básicos de:

- Estruturação de páginas web
- Organização de conteúdo
- Criação de landing pages
- Apresentação de produtos
